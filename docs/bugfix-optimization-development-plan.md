目标：把当前 5 模块改成 6 模块，并让旧的总经理办公室、晨会、待办、评级、徽章、利润体检进入 `dashboard` 首页。

### Phase 2：核心 UI 组件与消息中心

目标：补 Tooltip 系统，并把右侧消息中心和收件箱改成可筛选、可折叠、可按天查看。

- 新增通用 `Tooltip` 组件。

### Phase 3：财务与汇票增强

目标：把资金风险从“看得到”升级为“能处理”。

- `draft` tab 增强汇票列表、到期预警、逾期还款。

- 财务费用独立展示：手续费、罚息、贷款利息、浮库资金占用。
- 修复阶段 6 中汇票逾期授信释放、罚息基数和罚息累积相关 bug。

### Phase 4：市场竞争机制

目标：补齐本品价格战、默契定价、背刺、关系修复与竞品反制。

- 实现 `priceWarCount` 的实际触发和惩罚。
- 本品店关系值影响默契定价底线。
- 联合抗竞期间加入背刺事件。
- 增加关系修复按钮与冷却。
- 增加补贴申请、反挖人、价格战宣言、提升 CSI 等反制措施。

### Phase 5：进阶体验与长期成长

目标：补个人财富成就、投资人月审、士气影响、年度授信和趋势图。

- 市场份额改成 SVG 饼图。
- GM 士气影响全店效率。
- 年底自动评估汇票授信额度。
- 趋势图升级为 12 个月多指标。

### Phase 6：统一 bug 修复与验收

目标：集中修复提示词列出的 Critical/High/Medium/Low bug，并做长局基础验收。

- 资产负债表恒等，补充所有者权益。

- 构建、lint、浏览器导航验收。
