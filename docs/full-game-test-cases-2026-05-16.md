## 用例清单

| ID | 模块 | 测试目的 | 操作步骤 | 预期结果 |
| --- | --- | --- | --- | --- |
| FT-01 | 开局设置 | 验证地区、投资人、剧本、难度能组合开局 | 进入首页，切换开局选项，点击“开始接任” | 进入经营看板，日期、现金、目标和新手引导展示正常 |
| FT-02 | 导航与懒加载 | 验证所有模块入口可打开 | 逐一进入首页、销售运营、利润中心、组织人事、市场诊断、衍生中心下所有 tab | 页面无空白、无控制台错误，懒加载 chunk 可加载 |
| FT-03 | 厂家订货 | 验证订货配置和付款方式 | 在厂家订货页选择车型，打开配置弹窗，确认下单 | 生成在途订单，现金、融资或汇票状态按付款方式变化 |
| FT-04 | 到货与库存 | 验证日结推进在途订单、布展和库房 | 推进经营日直到订单到货，切到展厅定价执行布展/入库 | 库存数量、展厅/库房位置和日志同步更新 |
| FT-05 | 展厅经营 | 验证定价、补贴、试驾车和批售 | 调整车型价格，执行补贴、设为试驾车、批售或自动布展 | 对应库存、成本、试驾状态和经营日志正确变化 |
| FT-06 | 营销漏斗 | 验证营销预算、活动和线索产出 | 在漏斗营销页发起活动，推进日结 | 活动消耗预算并在后续日结产出线索、客流或销售机会 |
| FT-07 | 客户谈判 | 验证重点客户报价和成交路径 | 在客户谈判页对客户选择守价、均衡、金融、快速成交或放弃 | 成交/战败/审批结果进入日志、库存、GP 和客户档案 |
| FT-08 | 机会池与 CRM | 验证线索机会和客户跟进 | 在机会池执行邀约、报价、跟进，客户中心执行回访/转介绍等动作 | 机会状态转移，成功或关闭后沉淀历史/客户档案 |
| FT-09 | 审批中心 | 验证低价审批、谈判和客诉 | 在经营看板处理待审批事项 | 处理结果改变现金、成交率、CSI 或日志 |
| FT-10 | 经营事件 | 验证事件产生、处理和逾期 | 进入事件中心处理事件，或推进到逾期 | 成功/失败/逾期影响按配置结算，收件箱和日志可追踪 |
| FT-11 | 员工管理 | 验证招聘、培训、留任和压力 | 招聘 DCC/销售/客服/主播/技师，执行培训和留任 | 员工数量、能力、压力、忠诚度、薪酬成本正常变化 |
| FT-12 | 设施与门店 | 验证设施升级和门店可视化 | 升级设施，查看门店总览和蓝图楼层 | 展位/库容/工位等指标更新，蓝图坐标和员工浮层正常 |
| FT-13 | 财务与 GM 办公室 | 验证薪酬、垫资、谈判和报表 | 调整总经理薪酬，尝试个人垫资和授权谈判，查看财务报表 | 现金、个人账户、投资人关系、财务三表展示正常 |
| FT-14 | 返利与月结 | 验证厂家任务、返利和月报 | 推进至月末并完成月结，查看返利与月报 | 月结弹窗出现，返利/利润/投资人评分和历史报表生成 |
| FT-15 | 汇票与融资 | 验证承兑汇票和银行信用 | 使用汇票订货，推进到到期或逾期 | 汇票列表、兑付、逾期罚息和银行信用变化正常 |
| FT-16 | 市场与竞品 | 验证同城市场、价格战和反制 | 查看本地市场，执行竞品反制，推进日结/月结 | 市场份额、竞品状态、价格战和报告变化可见 |
| FT-17 | CSI 与售后 | 验证满意度、投诉和售后服务 | 执行关怀/回访/售后技师相关操作，推进日结 | CSI、投诉、售后收入和服务能力联动正常 |
| FT-18 | 衍生业务与二手车 | 验证金融精品策略、二手车建设和零售 | 切换衍生策略，建设/升级二手车展厅，整备、零售或批售二手车 | GP3、库存、现金和二手车状态正常变化 |

| FT-20 | 结局与重开 | 验证剧本目标、破产/解聘/优秀结局触发 | 自动化覆盖结局判定；人工确认结局弹窗入口和重开按钮 | 结局摘要可查看，重开回到开局配置 |

## 通过标准

- 自动化命令全部通过。
- 首屏、主要 tab、弹窗和懒加载页面不出现白屏或资源加载失败。
- 主经营循环能完成“开局 -> 订货 -> 到货 -> 布展 -> 线索/客户 -> 成交/事件 -> 月结 -> 存档”。
- 未发现阻断玩法继续的 P0/P1 问题。
- 若发现非阻断问题，测试报告必须记录复现步骤和建议归属模块。
